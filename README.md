# STM32_Simple_VCU

基于 STM32F103 的简易 VCU（Vehicle Control Unit，整车控制器）实验项目。

本项目用于学习嵌入式开发、ADC 采集、PWM 电机控制、状态机、CAN 通信、故障诊断和模块化软件设计。  
项目会按照 Demo 版本逐步迭代，从基础油门采集和电机控制开始，逐步扩展为一个简易 VCU 系统。

---

## 项目目标

- 使用 STM32 实现简易 VCU 控制逻辑
- 使用电位器模拟油门踏板输入
- 使用 PWM 控制电机转速
- 使用 OLED 显示油门开度和系统状态
- 后续加入刹车、钥匙、档位、故障诊断、CAN 通信等模块
- 记录每个 Demo 版本的开发过程

---

## 当前版本

当前版本：**Demo v1.0**

### Demo v1.0 - 油门 ADC 控制 PWM 电机

本版本实现了：

- PA0 ADC 采集油门模拟量
- ADC 数值转换为 0~100% 油门开度
- TIM2_CH3 输出 PWM
- PA4 / PA5 控制电机方向
- OLED 显示油门百分比
- 电机根据油门开度调节转速

控制流程：

```text
电位器 / 油门踏板
        ↓
STM32 ADC 采集
        ↓
油门开度 0~100%
        ↓
PWM 输出
        ↓
电机驱动模块
        ↓
电机转速变化
