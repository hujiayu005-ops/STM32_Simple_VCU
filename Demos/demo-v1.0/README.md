# Demo v1.0 - 油门 ADC 控制 PWM 电机

## 版本目标

本版本实现基于 STM32F103C8T6 的油门输入与电机 PWM 控制。

通过 PA0 采集电位器模拟油门信号，将 ADC 采集值转换为 0~100% 油门开度，并通过 TIM2_CH3 输出 PWM 控制电机转速。

## 已实现功能

- PA0 ADC 采集油门模拟量
- ADC 数值转换为 0~100% 油门开度
- TIM2_CH3 输出 PWM
- PA4 / PA5 控制电机方向
- OLED 显示油门百分比
- 电机根据油门开度调速

| 文件 | 功能 |
|---|---|
| User/main.c | 主程序 |
| User/油门.c | 油门 ADC 采集 |

| User/PWM.c | PWM 输出 |
| User/PWM.h | PWM 头文件 |
| User/电机.c | 电机控制 |
| User/电机.h | 电机控制头文件 |

## 硬件连接

| 功能 | STM32 引脚 | 说明 |
|---|---|---|
| 油门输入 | PA0 | ADC1_IN0 |
| PWM 输出 | PA2 | TIM2_CH3 |
| 电机方向 AIN1 | PA4 | 方向控制 |
| 电机方向 AIN2 | PA5 | 方向控制 |
| OLED 显示 | I2C / 软件 I2C | 显示油门开度 |
